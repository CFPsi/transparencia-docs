# Finanças

Cada documento financeiro pode ser categorizado em uma das opções abaixo:

* Balanço Financeiro
* Balanço Orçamentário
* Balanço Patrimonial
* Fluxo de Caixa
* Notas Explicativas
* Relação de Bens
* Variações Patrimoniais

Para incluir um novo documento financeiro siga os passos:

1. [Acesse o sistema](../../introducao/acessando_o_sistema.md)
2. No menu do painel clique no item **Finanças**
3. Clique no botão **Adicionar Novo**
4. Preencha o **Título** e o **Conteúdo**
5. Na caixa **Arquivos** (onde serão anexados os documentos) clique no botão **Adicionar Documento**
6. Na caixa **Documento** preencha o **Título** e adicione o arquivo (em pdf) clicando no botão **Adicionar Arquivo**, na proxima tela clique no botão **Selecionar arquivos**, depois que escolher o arquivo clique no botão **Inserir no post**
7. Para adicionar mais arquivos repita os passos 5 e 6 quantas vezes forem necessárias
8. Na caixa **Publicar** edite a data de publicação clicando no link **Editar** (que fica na frente do texto 'Publicar imediatamente') para o seu correto arquivamento
9. Na caixa **Demonstrativos** escolha uma opção e para finalizar clique no botão **Publicar**

**DICA**: Para colar o texto sem formatação ative o botão **Colar como texto** antes de colar.\
**Atenção:** Não deixe apenas números no campo Link permanente, pois isso pode causar um erro ao visualizar.

## Prestação de Contas

A Prestação de Contas é compreendida pelos seguintes itens:

* Comparativo de Despesa
* Comparativo de Receita
* Relação de pagamentos

A alimentação desses itens, necessitam dos aquivos (no formato csv) exportados do SISCONT.NET. Para aprender como alimentar essas áreas [consulte o artigo Prestação de Contas](https://oliveiraivan.gitbooks.io/transparencia/content/financas/prestacao-de-contas.html).
