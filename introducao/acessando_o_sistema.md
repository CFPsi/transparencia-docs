# Acessando o sistema

Após receber o **login** e **senha** para acessar o sistema, basta seguir as orientações abaixo.

1. Acesse o site [transparencia.cfp.org.br/crp\[número\]](http://transparencia.cfp.org.br) e navegue até o rodapé do site
2. Clique no link **Área restrita**
3. Faça o login utilizando seu **usuário** e **senha**.
